# Privacy Policy — Programmers Notifier

Last updated: 2026-04-20

## 수집하는 정보

Programmers Notifier는 다음 정보를 사용자의 기기 로컬에만 저장합니다:

- 디스코드 웹훅 URL
- 표시 이름 (선택 입력)
- 알림 설정 (성공만 전송, 문제 링크 포함 여부)
- 최근 전송 기록 (최대 50건)

## 수집하지 않는 정보

- 개인 식별 정보
- 작성한 코드
- 브라우저 방문 기록

## 데이터 사용 방법

저장된 데이터는 오직 사용자가 설정한 디스코드 웹훅으로 채점 결과를 전송하는 목적으로만 사용됩니다. 외부 서버로 데이터를 전송하거나 제3자와 공유하지 않습니다.

## 데이터 저장 위치

모든 데이터는 Chrome의 `chrome.storage.local` API를 통해 사용자의 기기에만 저장됩니다.
